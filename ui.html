Container Script: Beta

Programatically assign URLs to containers.

How to Use ContainerScript

Write JavaScript code to define container rules. The code runs each time you open a URL, with the url variable (an instance of the URL class) available in scope. The code is executed using Sval due to CSP restrictions.

Basic Usage

Return a string to open the URL in that container:

if (url.hostname === 'amazon.com') { return 'shopping'; }

Advanced Configuration

Return an object for more control over the container:

if (url.includes('bank.com')) return { name: "finance", icon: "fingerprint", color: "blue" };
